# メニュー刷新：ゲームイメージ画像つきメニュー

## 今回やること

メニュー画面を、各ゲームの「イメージ画像（サムネイル）」が表示されるカードグリッド型に刷新する。

- 画像は**コードで自動描画**する（本作の方針どおり画像ファイルは使わない）。
  各ゲームらしいミニ絵を pygame プリミティブで描く。
  - DONKEY KONG / DONKEY KONG '81: 斜め鉄骨・はしご・コング・樽
  - TETRIS: 積み上がったブロックと落下ミノ
  - COMING SOON（PAC-MAN / SPACE INVADERS）: プレースホルダ風の絵
- レイアウトは**カードのグリッド**。各カード＝サムネイル＋タイトル。
- 選択中のカードを強調（枠の発光・拡大など）。準備中は暗く「COMING SOON」。
- 既存の操作感を維持しつつ上下左右で選択できるようにする（グリッドなので左右移動も対応）。

## 受け入れ条件

- [x] メニューに各ゲームのサムネイル画像が表示される
- [x] カードグリッドで並び、選択中カードが明確に強調される
- [x] 矢印キーで選択移動、Enter/Space で決定、遊べるゲームに遷移できる
- [x] 準備中ゲームは選択しても遷移しない（COMING SOON 表示）
- [x] 実際に起動して見た目・操作を確認する
