# Phase 1: メニュー + ドンキーコング ステージ 1 実装

## 概要

レトロゲームセンター の MVP（最小限で動くもの）を実装。メニュー画面からドンキーコングを選んでプレイし、クリア or ゲームオーバーまで遊べる状態にする。

## 今回やること

1. **Pygame 環境のセットアップ**
   - Python 3.9+ 環境構築
   - Pygame, pytmx ライブラリのインストール
   - src/ 配下に基本ファイル構成を作成

2. **メニューシーン**
   - 800×600 ピクセル画面にゲーム一覧を表示
   - 矢印キーでゲーム選択、Enter で決定
   - ドンキーコングを選択可能に

3. **ドンキーコング ゲームシーン**
   - Tiled で作成したマップを読み込み・表示
   - プレイヤーキャラクターを画面左に配置
   - 左右キーで移動、スペースキーでジャンプ
   - 樽が定時間隔で生成・移動
   - プレイヤーと樽の衝突判定（ゲームオーバー）
   - ゴール地点への到達判定（クリア）

4. **ゲームオーバー / クリアシーン**
   - 結果表示画面
   - スコア表示（固定値）
   - Enterキーでメニューに戻る
