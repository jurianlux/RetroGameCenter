# 新規ゲーム追加: テトリス

## 概要

レトロゲームセンターに新しいゲーム「テトリス」を追加する。メニューから「TETRIS」を
選んでプレイでき、ブロックが積み上がってゲームオーバーになるまで遊べる状態にする。
既存のドンキーコングと同じシーン構成・実装パターン（BaseScene / SceneManager）に
合わせる。

## スコープ（ヒアリング結果）

- 基本7種テトリミノ（I/O/T/S/Z/J/L）＋回転
- ネクスト表示（次に落ちてくるブロックをプレイ画面横に表示）
- レベル/速度上昇＋スコア（消去ライン数に応じてレベルアップ、落下速度が増す）
- ゲームオーバーは**テトリス内で完結**（その場でリスタート / メニュー選択）
- ホールド機能は**含めない**

## 今回やること

1. **テトリスシーン（TetrisScene）**
   - 10列 × 20行のフィールドを画面中央に描画
   - 7種テトリミノの落下・自動降下（重力タイマー）
   - 左右キーで移動、上キー（または X/Z）で回転、下キーでソフトドロップ
   - スペースキーでハードドロップ
   - 着地でフィールドに固定、揃ったラインを消去
   - ライン消去数に応じたスコア加算・レベルアップ・落下速度上昇
   - 新しいブロックが出現位置に置けなくなったらゲームオーバー

2. **ネクスト表示・HUD**
   - SCORE / LEVEL / LINES をサイドに表示
   - NEXT（次のテトリミノ）をサイドに表示

3. **ゲームオーバー処理（シーン内完結）**
   - フィールド上にゲームオーバー表示
   - Enter / R でその場でリスタート、Esc でメニューへ戻る（Esc は既存の共通仕様）

4. **メニューへの組み込み**
   - `menu_scene.py` の GAMES に「TETRIS」を追加（選択可能に）
   - `main.py` に TetrisScene を登録

## やらないこと（スコープ外）

- ホールド機能
- ゴースト（着地予測）表示
- SRS の壁蹴り（回転補正）の厳密再現 —— 基本的な回転＋簡易の押し戻しのみ
- T-spin など特殊得点
- サウンド（BGM・SE）
- スコアランキングの永続保存（JSON）
