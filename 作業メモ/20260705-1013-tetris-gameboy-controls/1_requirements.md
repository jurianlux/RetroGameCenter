# テトリス操作方法の変更（初代ゲームボーイ準拠）

## 今回やること

テトリスの操作方法が「ファミコン（NES）版」相当になっていたため、
**初代ゲームボーイ版**の操作に変更する。あわせて画面内の操作説明を更新する。

### 操作の対応表（変更後）

| キー | 動作 | ゲームボーイのボタン |
|------|------|----------------------|
| ← → | 左右移動 | 十字キー左右 |
| ↓ | ソフトドロップ | 十字キー下 |
| X | 右回転 | A ボタン |
| Z | 左回転 | B ボタン |

### ファミコン版からの主な違い

- ↑（上キー）での回転を廃止（ゲームボーイでは上キーは未使用）
- 左右2方向の回転に対応（A=右回転 / B=左回転）
- ハードドロップ（SPACE）を削除（初代ゲームボーイ版には無い機能）

## 受け入れ条件

- [x] X で右回転、Z で左回転ができる
- [x] ↑キーでは回転しない
- [x] SPACE（ハードドロップ）が機能しない（削除済み）
- [x] ← → 移動、↓ ソフトドロップは従来どおり動作する
- [x] 画面内に操作説明がコンパクトに表示される（CONTROLS 見出し付き）
- [x] 起動・実行時にエラーが出ない

## やらないこと（スコープ外）

- キー割り当てのカスタマイズ機能（X/Z 固定）
- ホールド機能・ゴースト表示など他機能の追加
- ドンキーコングなど他ゲームの操作変更

## タスクリスト

- [x] `tetris_scene.py` の入力処理を GB 準拠に変更（X=右回転 / Z=左回転）
- [x] ハードドロップ（`_hard_drop`）と SPACE / ↑ 入力を削除
- [x] `tetromino.py` の `next_rotation` に回転方向引数を追加
- [x] 画面内の操作説明（CONTROLS）を更新
- [x] ヘッドレスで回転・キー入力の実挙動を検証

## 変更ファイル

- `src/scenes/tetris_scene.py`
- `src/game_objects/tetromino.py`

# 追記：ソフトドロップ長押し対応と効果音（同日）

## 今回やること

1. **下キー長押しでソフトドロップ継続** — 押している間ずっと落下し続けるようにする
   （従来は KEYDOWN 単発で、1回押すごとに1マスだけ落ちていた）
2. **効果音の追加** — 操作・進行に合わせて SE を鳴らす

### 効果音の対応表

| 場面 | SE 名 | イメージ |
|------|-------|----------|
| 左右移動 | move | コッ |
| 回転（X / Z） | rotate | ピッ |
| ピース設置 | lock | トッ |
| ライン消去 | line | ピロリン |
| レベルアップ | levelup | ファンッ |
| ゲームオーバー | death | 下降スイープ |

## 実装メモ

- 長押しは `update()` 内で `pygame.key.get_pressed()` を毎フレーム確認し、
  `SOFT_DROP_INTERVAL`（0.05秒）ごとに1マス落下。押した瞬間の即応も維持。
- 効果音は既存の `utils/synth_audio.py`（波形合成・音声ファイル不要）を再利用。
  DK81 と同じ `SoundBank` を使用。合成・再生失敗時は無音で継続する既存ポリシーを踏襲。

## 受け入れ条件

- [x] 下キーを押している間、連続してソフトドロップする
- [x] 下キーを離すと落下が止まる
- [x] 移動・回転・設置・ライン消去・レベルアップ・ゲームオーバーで SE が鳴る
- [x] 音声が無効な環境でもエラーなく動作する（無音で継続）
- [x] ヘッドレスで長押し落下と SE 合成の実挙動を検証

## やらないこと（スコープ外）

- BGM の追加（synth_audio の BGM は現状スタブのまま）
- SE 音量・音色の細かなチューニング（要望が出たら対応）

## タスクリスト

- [x] `synth_audio.py` に move / rotate / lock / line / levelup の SE を追加
- [x] `tetris_scene.py` の `update()` に下キー長押しソフトドロップを実装
- [x] 各操作・進行イベントに `play_se` を配線
- [x] 画面内ヒントを `v (HOLD): DROP` に更新
- [x] ヘッドレスで実挙動を検証（長押し0.30秒で6マス落下・SE9種合成成功を確認）
