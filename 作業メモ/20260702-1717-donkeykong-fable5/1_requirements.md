# 追加ゲーム: ドンキーコング（アーケード忠実版 / Fable 5 制作）

## 概要

既存のシンプルなドンキーコング（矩形・円の簡易図形、単面、無音）は**そのまま残す**。
本作は **オリジナル 1981 年アーケード版の 25m（樽）ステージを徹底的に再現・作り込んだ
高品質版**を、**新しいゲームとしてメニューに追加**する。実装は **Fable 5** が行い、
「Fable 5 でどこまでできるか」を 1 面に凝縮して示すことを目的とする。

- メニュー表示名: **`DONKEY KONG '81`**（既存 `DONKEY KONG` と区別）
- シーンキー: `donkey_kong_81` / クラス: `DonkeyKong81Scene`
- 既存ゲーム（donkey_kong / tetris）とメニュー・遷移は完全に共存する

## 進め方（承認済み・詳細は 2_design.md）

- **実装は新しいセッションで依頼**（本セッションでは着手しない）
- 本体実装は **Fable 5 サブエージェントに一括委譲**（承認済み仕様を丸ごと渡し、原則 1 発で実装）
- **Opus は最小関与**（オーケストレーション＋最終検証のみ）でプレミアム枠を温存
- **2 段マイルストーン**で進行し、枠が尽きても“遊べる物”が必ず残るようにする
  - **A（必達・遊べる版）**: ステージ／マリオ／樽／衝突／スコア・HUD／ミス→復帰→ゲームオーバー／クリア→clear／基本 SE／メニュー統合
  - **B（見せ場）**: ファイアボール／ハンマー／オイルドラム連動／イントロ&クリア演出／BGM 拡充／ドット絵作り込み／ハイスコア保存（任意）
- **A 完了後に使用量を実測**してから B に進むか判断する

## 今回やること

### 1. ステージ再現（25m 樽ステージ）
- オリジナル準拠の **7 段の斜め鉄骨（girder）** と**はしご**の配置
- 一部の**登り切れない「壊れたはしご」**（本家の特徴）
- 最下段左の**オイルドラム缶＋炎**
- 上部の **ドンキーコング**（樽を持ち上げ・投げる・胸を叩くアニメ）
- 上部の **ポーリン**（"HELP!"、揺れるアニメ）

### 2. プレイヤー（マリオ）
- 左右移動・はしご昇降・ジャンプ（斜面追従）
- **ドット絵風の描画**（帽子・オーバーオール・歩行アニメ・向き反転）
- ミス時の**やられアニメ**（回転落下）と復帰

### 3. 敵・ギミック
- **樽**: 斜面を転がる／確率ではしごを降りる／オイルドラムへ寄る挙動
- **ファイアボール（炎）**: 樽がオイルドラムに落ちると発生し、鉄骨・はしごを移動してマリオを追う
- **ハンマー**: 取得すると一定時間、樽・炎を破壊できる。取得中はジャンプ／はしご不可（本家仕様）

### 4. スコア・進行
- ジャンプで樽/炎を飛び越え **+100**、ハンマー破壊で加点
- **ボーナスタイマー**（時間経過で減少、クリア時に残りを加算）
- ライフ制、**ハイスコア表示**、`1UP`、`HOW HIGH?` の m 表示など HUD

### 5. 演出（Fable 5 の見せ場）
- **イントロ**: DK がポーリンを抱えて登り、踏みつけて鉄骨を曲げ、`HOW HIGH CAN YOU GET?` → `25 M`
- **クリア**: ハート出現 → DK 転落の簡易演出 → clear シーンへ
- **合成サウンド**: BGM（イントロ／ハンマー中など）＋ SE（ジャンプ・得点・ミス・ハンマー・クリア）
  - 音声ファイルは持たず、**標準ライブラリでメモリ上に波形合成**して再生（numpy 不使用 = 新規依存なし。ARCH-004「追加ライブラリ不要」を維持）
  - mixer 初期化に失敗する環境では**無音で正常動作**（クラッシュしない）

### 6. 統合（既存を壊さない）
- 新規ファイル中心で実装（新シーン＋本作専用の game_objects モジュール）
- 共有ファイルへの変更は**追記のみ**に限定:
  - `config.py`: 本作用の定数を追記
  - `scenes/menu_scene.py`: `GAMES` に 1 エントリ追加
  - `main.py`: 新シーンを register
- 既存 `player.py` / `barrel.py` / `level.py` は**変更しない**（既存 DK が壊れないこと）

## 受け入れ条件

- [ ] メニューに新エントリが増え、既存の DONKEY KONG / TETRIS もこれまで通り遊べる
- [ ] 新ゲーム選択 → イントロ演出 → プレイ開始まで通しで動く
- [ ] マリオが移動・はしご昇降・ジャンプでき、斜面に追従する
- [ ] 樽が斜面を転がり、確率ではしごを降り、オイルドラムへ寄る
- [ ] オイルドラムから炎が発生し、マリオを追う
- [ ] ハンマー取得で樽・炎を破壊でき、取得中はジャンプ/はしごが制限される
- [ ] 樽/炎を飛び越えると +100、ボーナスタイマーが減少する
- [ ] 被弾でミス（やられアニメ→復帰）、ライフ 0 で game_over シーンへ
- [ ] 最上段ポーリン到達でクリア演出 → clear シーンへ
- [ ] BGM・SE が鳴る（無音環境でもクラッシュしない）
- [ ] ESC でメニューに戻れる／既存の遷移を壊さない
- [ ] `python src/main.py` がエラーなく起動する

## やらないこと（スコープ外）

- 50m コンベア / 75m エレベーター / 100m リベットの各ステージ（本作は 25m のみ）
- 既存ドンキーコング（`donkey_kong`）の改修・置き換え
- ネットワーク機能・オンラインランキング（ハイスコアはローカル JSON、実装は任意）
- 外部の音声/画像アセットファイルの追加（すべてコード生成）
